hoverIntent

A jQuery Plugin

Presented by Andrew Lancaster

What is hoverIntent?

A jQuery plugin that extends the jQuery.hover method to better predict the user's intentions when moving the mouse cursor around the screen.

Why use it?

  • Keep events from firing unintentionally
  • Lots of settings
    • Handler In and Out functions
    • Timeout
    • Selector
    • Sensitivity
    • Interval

jQuery.hover Code


$(document).ready(function(){
	$("ul > li").hover(slideDown, slideUp);
});

function slideDown(){$(this).find("ul").stop().slideDown(200);}
function slideUp(){$(this).find("ul").stop().slideUp(200);}

Live Example

hoverIntent Code


$(document).ready(function(){

	$("ul").hoverIntent({
	    over: slideDown,
	    out: slideUp,
	    selector: 'li',
	    timeout: 150,
	    sensitivity: 3
	});

});

function slideDown(){$(this).find("ul").slideDown(200);}
function slideUp(){$(this).find("ul").slideUp(200);}
