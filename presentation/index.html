hoverIntent

A jQuery Plugin

Presented by Andrew Lancaster

What is hoverIntent?

A jQuery plugin that extends the .hover jQuery method to better guess the user's intentions when moving the mouse cursor around the screen.

Why use it?

The main reason for this plugin is to keep events from firing when they aren't supposed to. Even though it can look pretty cool when you move your mouse over several navigation buttons and see dropdowns begin to move instantly, it could hurt the user's experience.

hoverIntent(less) Code


$(document).ready(function(){
	$("ul > li").hover(slideDown, slideUp);
});

function slideDown(){$(this).find("ul").stop().slideDown(200);}
function slideUp(){$(this).find("ul").stop().slideUp(200);}

hoverIntent Code


$(document).ready(function(){

	$("ul").hoverIntent({
	    over: slideDown,
	    out: slideUp,
	    selector: 'li',
	    timeout: 150,
	    sensitivity: 3
	});

});

function slideDown(){$(this).find("ul").slideDown(200);}
function slideUp(){$(this).find("ul").slideUp(200);}

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.
